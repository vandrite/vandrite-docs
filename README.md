## ğŸ Getting Started

Clone the repository and install dependencies to get started locally.

```bash
git clone https://github.com/vandrite/vandrite-docs.git
cd vandrite-docs
npm install
# or
pnpm install
```

### Development Server

Start the development server to preview changes instantly:

```bash
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the documentation site.
