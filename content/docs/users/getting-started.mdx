# Getting Started

Welcome to Vandrite! This guide will help you get started with the app.

## Installation

Download Vandrite from [vandrite.com](https://vandrite.com) for your operating system:

- Windows
- macOS
- Linux

## First Steps

After installing Vandrite, you can:

1. **Create your first note** - Start writing with our powerful markdown editor
2. **Set up a vault** - Organize your notes into a folder structure
3. **Explore features** - Try out Kanban boards, Journal, and Canvas
4. **Install plugins** - Extend Vandrite with community plugins
